# Odin Recipes

## Project Description
This is my first basic HTML website project from The Odin Project. The goal of this project is to practice creating a simple website with structural HTML elements, links, and images.

## Skills Demonstrated / Lessons Learned
* Structuring an HTML document correctly
* Using headings (`<h1>` through `<h6>`), paragraphs, and lists
* Working with absolute and relative file paths for links and images
* Utilizing Git and GitHub for version control
